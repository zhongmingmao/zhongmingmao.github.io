MCP Docs - Overview

Created2026-03-03|AIMCP
|Word Count:176|Reading Time:1mins

Introduction

  1. MCP (Model Context Protocol) is an open-source standard for connecting AI applications to external systems.
  2. Using MCP, AI applications like Claude or ChatGPT can connect to
    • data sources (e.g. local files, databases)
    • tools (e.g. search engines, calculators)
    • workflows (e.g. specialized prompts)—enabling them to access key information and perform tasks.
  3. Think of MCP like a USB-C port for AI applications.
    • Just as USB-C provides a standardized way to connect electronic devices
    • MCP provides a standardized way to connect AI applications to external systems.

Why does MCP matter?

Depending on where you sit in the ecosystem, MCP can have a range of benefits.

  1. Developers
    • MCP reduces development time and complexity when building, or integrating with, an AI application or agent.
  2. AI applications or agents
    • MCP provides access to an ecosystem of data sources, tools and apps which will enhance capabilities and improve the end-user experience.
  3. End-users
    • MCP results in more capable AI applications or agents which can access your data and take actions on your behalf when necessary.
