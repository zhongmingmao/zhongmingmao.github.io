жӣҙж–°иҜӯеҸҘ

жү§иЎҢиҝҮзЁӢ

  1. йҖҡиҝҮиҝһжҺҘеҷЁпјҢе®ўжҲ·з«ҜдёҺMySQLе»әз«ӢиҝһжҺҘ
  2. updateиҜӯеҸҘдјҡжҠҠTиЎЁдёҠзҡ„жүҖжңүжҹҘиҜўзј“еӯҳжё…з©ә
  3. еҲҶжһҗеҷЁдјҡйҖҡиҝҮиҜҚжі•еҲҶжһҗе’ҢиҜӯжі•еҲҶжһҗиҜҶеҲ«иҝҷжҳҜдёҖжқЎжӣҙж–°иҜӯеҸҘ
  4. дјҳеҢ–еҷЁдјҡеҶіе®ҡдҪҝз”ЁidиҝҷдёӘзҙўеј•пјҲиҒҡз°Үзҙўеј•пјү
  5. жү§иЎҢеҷЁиҙҹиҙЈе…·дҪ“жү§иЎҢпјҢжүҫеҲ°еҢ№й…Қзҡ„дёҖиЎҢпјҢз„¶еҗҺжӣҙж–°
  6. жӣҙж–°иҝҮзЁӢдёӯиҝҳдјҡж¶үеҸҠredologпјҲйҮҚеҒҡж—Ҙеҝ—пјүе’ҢbinlogпјҲеҪ’жЎЈж—Ҙеҝ—пјүзҡ„ж“ҚдҪң

redolog вҖ“ InnoDB

  1. еҰӮжһңжҜҸж¬Ўжӣҙж–°ж“ҚдҪңйғҪйңҖиҰҒзӣҙжҺҘеҶҷе…ҘзЈҒзӣҳпјҲеңЁзЈҒзӣҳдёӯжүҫеҲ°зӣёе…ізҡ„и®°еҪ•е№¶жӣҙж–°пјүпјҢж•ҙдёӘиҝҮзЁӢзҡ„IOжҲҗжң¬е’ҢжҹҘжүҫжҲҗжң¬йғҪеҫҲй«ҳ
  2. й’ҲеҜ№иҝҷз§Қжғ…еҶөпјҢMySQLйҮҮз”Ёзҡ„жҳҜWALжҠҖжңҜпјҲWrite-Ahead Loggingпјүпјҡ_е…ҲеҶҷж—Ҙеҝ—пјҢеҶҚеҶҷзЈҒзӣҳ_
  3. еҪ“жңүдёҖжқЎи®°еҪ•йңҖиҰҒжӣҙж–°зҡ„ж—¶еҖҷпјҢInnoDBдјҡе…ҲжҠҠи®°еҪ•еҶҷеҲ°redologпјҲredolog bufferпјүпјҢе№¶жӣҙж–°еҶ…еӯҳпјҲbuffer poolпјү
    • InnoDBдјҡеңЁйҖӮеҪ“зҡ„ж—¶еҖҷпјҲдҫӢеҰӮзі»з»ҹз©әй—ІпјүпјҢе°ҶиҝҷдёӘж“ҚдҪңи®°еҪ•еҲ°зЈҒзӣҳйҮҢйқўпјҲеҲ·и„ҸйЎөпјү
  4. InnoDBзҡ„redologжҳҜеӣәе®ҡеӨ§е°Ҹзҡ„пјҢеҰӮжһңжҜҸдёӘж—Ҙеҝ—ж–Үд»¶еӨ§е°Ҹдёә1GBпјҢ4дёӘж—Ҙеҝ—ж–Үд»¶дёәдёҖз»„
    • redologзҡ„жҖ»еӨ§е°Ҹдёә4GBпјҢ_еҫӘзҺҜеҶҷ_
    • write posжҳҜеҪ“еүҚи®°еҪ•зҡ„дҪҚзҪ®пјҢдёҖиҫ№еҶҷдёҖиҫ№еҗҺз§»пјҢеҶҷеҲ°3еҸ·ж–Үд»¶жң«е°ҫеҗҺе°ұеӣһеҲ°0еҸ·ж–Үд»¶ејҖеӨҙ
      • redologжҳҜйЎәеәҸеҶҷпјҢж•°жҚ®ж–Үд»¶жҳҜйҡҸжңәеҶҷ
    • checkpointжҳҜеҪ“еүҚиҰҒж“ҰйҷӨзҡ„дҪҚзҪ®пјҢж“ҰйҷӨи®°еҪ•еүҚйңҖиҰҒе…ҲжҠҠеҜ№еә”зҡ„ж•°жҚ®иҗҪзӣҳпјҲжӣҙж–°еҶ…еӯҳйЎөпјҢзӯүеҫ…еҲ·и„ҸйЎөпјү
    • write posеҲ°checkpointд№Ӣй—ҙзҡ„йғЁеҲҶеҸҜд»Ҙз”ЁжқҘи®°еҪ•ж–°зҡ„ж“ҚдҪң
      • еҰӮжһңwrite posиө¶дёҠдәҶcheckpointпјҢиҜҙжҳҺredologе·Іж»ЎпјҢдёҚиғҪеҶҚжү§иЎҢж–°зҡ„жӣҙж–°ж“ҚдҪңпјҢйңҖиҰҒе…ҲжҺЁиҝӣcheckpoint
      • еҸӘиҰҒwrite posжңӘиө¶дёҠcheckpointпјҢе°ұеҸҜд»Ҙжү§иЎҢж–°зҡ„жӣҙж–°ж“ҚдҪң
    • checkpointеҲ°write posд№Ӣй—ҙзҡ„йғЁеҲҶзӯүеҫ…иҗҪзӣҳпјҲе…Ҳжӣҙж–°еҶ…еӯҳйЎөпјҢз„¶еҗҺзӯүеҫ…еҲ·и„ҸйЎөпјү
      • еҰӮжһңcheckpointиө¶дёҠдәҶwrite posпјҢиҜҙжҳҺredologе·Із©ә
  5. жңүдәҶredologд№ӢеҗҺпјҢInnoDBиғҪдҝқиҜҒж•°жҚ®еә“еҚідҪҝеҸ‘з”ҹејӮеёёйҮҚеҗҜпјҢд№ӢеүҚжҸҗдәӨзҡ„и®°еҪ•йғҪдёҚдјҡдёўеӨұпјҢиҫҫеҲ°crash-safe
  6. еҰӮжһңredologеӨӘе°ҸпјҢдјҡеҜјиҮҙеҫҲеҝ«иў«еҶҷж»ЎпјҢз„¶еҗҺе°ұдёҚеҫ—дёҚејәиЎҢеҲ·redologпјҢиҝҷж ·WALжңәеҲ¶зҡ„иғҪеҠӣе°ұж— жі•еҸ‘жҢҘеҮәжқҘ
    • еҰӮжһңзЈҒзӣҳиғҪиҫҫеҲ°еҮ TBпјҢйӮЈд№ҲеҸҜд»Ҙе°Ҷredologи®ҫзҪ®4дёӘдёҖз»„пјҢжҜҸдёӘж—Ҙеҝ—ж–Үд»¶еӨ§е°Ҹдёә1GB

ж•°жҚ®иҗҪзӣҳ

  1. redologе№¶жІЎжңүи®°еҪ•ж•°жҚ®йЎөзҡ„е®Ңж•ҙи®°еҪ•пјҢеҸӘжҳҜи®°еҪ•ж•°жҚ®йЎөзҡ„еҸҳжӣҙпјҢеӣ жӯӨredologжң¬иә«е№¶жІЎжңүзӣҙжҺҘжӣҙж–°зЈҒзӣҳйЎөзҡ„иғҪеҠӣ
  2. MySQLе®һдҫӢжӯЈеёёиҝҗиЎҢпјҢеңЁеҶ…еӯҳдёӯзҡ„ж•°жҚ®йЎөиў«дҝ®ж”№еҗҺпјҢи·ҹзЈҒзӣҳдёҠзҡ„ж•°жҚ®йЎөдёҚдёҖиҮҙпјҢз§°дёәи„ҸйЎө
    • жңҖз»Ҳзҡ„ж•°жҚ®иҗҪзӣҳпјҢжҢҮзҡ„жҳҜжҠҠеҶ…еӯҳдёӯзҡ„ж•°жҚ®йЎөиҰҶзӣ–зЈҒзӣҳдёҠзҡ„ж•°жҚ®йЎөпјҢиҝҷдёӘиҝҮзЁӢдёҺredologж— е…і
  3. еңЁеҙ©жәғжҒўеӨҚзҡ„еңәжҷҜпјҢInnoDBеҰӮжһңеҲӨж–ӯжҳҜдёӢйқўзҡ„2.aеңәжҷҜ
    • е°ұдјҡиҜ»еҸ–зЈҒзӣҳдёҠеҜ№еә”зҡ„ж•°жҚ®йЎөеҲ°еҶ…еӯҳдёӯпјҢз„¶еҗҺеә”з”ЁredologпјҢжӣҙж–°еҶ…еӯҳйЎө
    • жӣҙж–°е®ҢжҲҗеҗҺпјҢиҝҷдёӘеҶ…еӯҳйЎөе°ұеҸҳжҲҗдәҶи„ҸйЎөпјҢзӯүеҫ…иў«еҲ·еҲ°зЈҒзӣҳдёҠ

redolog buffer

  1. redolog bufferз”ЁдәҺеӯҳж”ҫredolog
  2. жү§иЎҢе®Ңз¬¬дёҖдёӘINSERTеҗҺпјҢеҶ…еӯҳдёӯпјҲbuffer poolпјүзҡ„ж•°жҚ®йЎөиў«дҝ®ж”№дәҶпјҢеҸҰеӨ–redolog bufferд№ҹеҶҷе…ҘдәҶж—Ҙеҝ—
  3. COMMITпјҡзңҹжӯЈжҠҠж—Ҙеҝ—еҶҷеҲ°redologпјҲib_logfileXпјү
  4. иҮӘеҠЁејҖеҗҜдәӢеҠЎзҡ„SQLиҜӯеҸҘйҡҗејҸеҢ…еҗ«дёҠиҝ°иҝҮзЁӢ

binlog вҖ“ Server

  1. redologжҳҜInnoDBзү№жңүзҡ„ж—Ҙеҝ—пјҢbinlogеұһдәҺServerеұӮж—Ҙеҝ—
  2. жңүдёӨд»Ҫж—Ҙеҝ—зҡ„еҺҶеҸІеҺҹеӣ 
    • дёҖејҖе§Ӣе№¶жІЎжңүInnoDBпјҢйҮҮз”Ёзҡ„жҳҜMyISAMпјҢдҪҶMyISAMжІЎжңүcrash-safeзҡ„иғҪеҠӣпјҢbinlogж—Ҙеҝ—еҸӘиғҪз”ЁдәҺеҪ’жЎЈ
    • InnoDBжҳҜд»ҘжҸ’д»¶зҡ„еҪўејҸеј•е…ҘMySQLзҡ„пјҢдёәдәҶе®һзҺ°crash-safeпјҢInnoDBйҮҮз”ЁдәҶredologзҡ„ж–№жЎҲ
  3. binlogдёҖејҖе§Ӣзҡ„и®ҫи®Ўе°ұжҳҜдёҚж”ҜжҢҒеҙ©жәғжҒўеӨҚпјҲеҺҹеә“пјүзҡ„пјҢеҰӮжһңдёҚиҖғиҷ‘жҗӯе»әд»Һеә“зӯүж“ҚдҪңпјҢbinlogжҳҜеҸҜд»Ҙе…ій—ӯзҡ„пјҲsql_log_binпјү
  4. redolog vs binlog
    • redologжҳҜInnoDBзү№жңүзҡ„пјҢbinlogжҳҜMySQLзҡ„ServerеұӮе®һзҺ°зҡ„пјҢжүҖжңүеұӮйғҪеҸҜд»ҘдҪҝз”Ё
    • redologжҳҜзү©зҗҶж—Ҙеҝ—пјҢи®°еҪ•жҹҗдёӘж•°жҚ®йЎөдёҠеҒҡдәҶд»Җд№Ҳдҝ®ж”№
      • binlogжҳҜйҖ»иҫ‘ж—Ҙеҝ—пјҢи®°еҪ•жҹҗдёӘиҜӯеҸҘзҡ„еҺҹе§ӢйҖ»иҫ‘
      • йҖ»иҫ‘ж—Ҙеҝ—пјҡжҸҗдҫӣз»ҷеҲ«зҡ„еј•ж“Һз”ЁпјҢжҳҜеӨ§е®¶йғҪиғҪзҗҶи§Јзҡ„йҖ»иҫ‘пјҢдҫӢеҰӮжҗӯе»әд»Һеә“
      • зү©зҗҶж—Ҙеҝ—пјҡеҸӘиғҪеҶ…йғЁдҪҝз”ЁпјҢе…¶д»–еј•ж“Һж— жі•е…ұдә«еҶ…йғЁзҡ„зү©зҗҶж јејҸ
    • redologжҳҜеҫӘзҺҜеҶҷпјҢз©әй—ҙеӣәе®ҡпјҢдёҚиғҪжҢҒд№…дҝқеӯҳпјҢжІЎжңүеҪ’жЎЈеҠҹиғҪ
      • binlogжҳҜиҝҪеҠ еҶҷпјҢз©әй—ҙдёҚеҸ—йҷҗеҲ¶пјҢжңүеҪ’жЎЈеҠҹиғҪ
    • redologдё»иҰҒз”ЁдәҺcrash-safeпјҢеҺҹеә“жҒўеӨҚ
      • binlogдё»иҰҒз”ЁдәҺжҒўеӨҚжҲҗдёҙж—¶еә“пјҲд»Һеә“пјү
    • еҙ©жәғжҒўеӨҚзҡ„иҝҮзЁӢдёҚеҶҷbinlogпјҲеҸҜиғҪйңҖиҰҒиҜ»binlogпјҢеҰӮжһңbinlogжңүжү“ејҖпјҢдёҖиҲ¬йғҪдјҡжү“ејҖпјү
      • з”ЁbinlogжҒўеӨҚе®һдҫӢпјҲд»Һеә“пјүпјҢйңҖиҰҒеҶҷredolog
  5. redologж”ҜжҢҒдәӢеҠЎзҡ„жҢҒд№…жҖ§пјҢundologж”ҜжҢҒдәӢеҠЎзҡ„йҡ”зҰ»жҖ§
  6. redologеҜ№еә”з”ЁејҖеҸ‘жқҘиҜҙжҳҜйҖҸжҳҺзҡ„
  7. binlogжңүдёӨз§ҚжЁЎејҸ
    • statementж јејҸпјҡSQLиҜӯеҸҘ
    • rowж јејҸпјҡиЎҢеҶ…е®№пјҲи®°дёӨжқЎпјҢжӣҙж–°еүҚе’Ңжӣҙж–°еҗҺпјүпјҢжҺЁиҚҗ
      • ж—Ҙеҝ—дёҖж ·зҡ„еҸҜд»Ҙз”ЁдәҺйҮҚж”ҫ

update еҶ…йғЁжөҒзЁӢ

жө…иүІжЎҶеңЁInnoDBеҶ…йғЁжү§иЎҢпјҢж·ұиүІжЎҶеңЁжү§иЎҢеҷЁдёӯжү§иЎҢ

  1. жү§иЎҢеҷЁе…ҲйҖҡиҝҮInnoDBиҺ·еҸ–id=2иҝҷдёҖиЎҢпјҢidжҳҜдё»й”®пјҢInnoDBеҸҜд»ҘйҖҡиҝҮиҒҡз°Үзҙўеј•жүҫеҲ°иҝҷдёҖиЎҢ
    • еҰӮжһңid=2иҝҷдёҖиЎҢжүҖеңЁзҡ„ж•°жҚ®йЎөжң¬жқҘе°ұеңЁеҶ…еӯҳпјҲInnoDB Buffer PoolпјүдёӯпјҢзӣҙжҺҘиҝ”еӣһз»ҷжү§иЎҢеҷЁ
    • еҗҰеҲҷе…Ҳд»ҺзЈҒзӣҳиҜ»е…ҘеҶ…еӯҳпјҢз„¶еҗҺеҶҚиҝ”еӣһ
  2. жү§иЎҢеҷЁжӢҝеҲ°InnoDBиҝ”еӣһзҡ„иЎҢж•°жҚ®пјҢиҝӣиЎҢ**+1**ж“ҚдҪңпјҢеҫ—еҲ°ж–°зҡ„дёҖиЎҢж•°жҚ®пјҢеҶҚи°ғз”ЁInnoDBзҡ„еј•ж“ҺжҺҘеҸЈеҶҷе…ҘиҝҷиЎҢж•°жҚ®
  3. InnoDBйҰ–е…Ҳе°ҶиҝҷиЎҢж–°ж•°жҚ®жӣҙж–°еҲ°еҶ…еӯҳпјҲInnoDB Buffer PoolпјүдёӯпјҢеҗҢж—¶е°ҶиҝҷдёӘжӣҙж–°ж“ҚдҪңи®°еҪ•еҲ°redologпјҲзү©зҗҶи®°еҪ•пјү
    • жӣҙж–°еҲ°еҶ…еӯҳдёӯпјҢеңЁдәӢеҠЎжҸҗдәӨеҗҺпјҢеҗҺз»ӯзҡ„жҹҘиҜўе°ұеҸҜд»ҘзӣҙжҺҘеңЁеҶ…еӯҳдёӯиҜ»еҸ–иҜҘж•°жҚ®йЎөпјҢдҪҶжӯӨж—¶зҡ„ж•°жҚ®еҸҜиғҪиҝҳжІЎжңүзңҹжӯЈиҗҪзӣҳ
      • дҪҶеңЁдәӢеҠЎжҸҗдәӨеүҚпјҢе…¶д»–дәӢеҠЎжҳҜж— жі•зңӢеҲ°иҝҷдёӘеҶ…еӯҳдҝ®ж”№зҡ„
      • иҖҢеңЁдәӢеҠЎжҸҗдәӨеҗҺпјҢиҜҙжҳҺе·Із»ҸжҲҗеҠҹеҶҷе…ҘдәҶredologпјҢеҸҜеҙ©жәғжҒўеӨҚпјҢдёҚдјҡдёўж•°жҚ®пјҢеӣ жӯӨеҸҜд»ҘзӣҙжҺҘиҜ»еҶ…еӯҳзҡ„ж•°жҚ®
    • еҲҡжӣҙж–°зҡ„еҶ…еӯҳжҳҜдёҚдјҡеҲ йҷӨзҡ„пјҢйҷӨйқһеҶ…еӯҳдёҚеӨҹз”ЁпјҢеңЁж•°жҚ®д»ҺеҶ…еӯҳеҲ йҷӨд№ӢеүҚпјҢзі»з»ҹдјҡдҝқиҜҒе®ғд»¬е·Із»ҸиҗҪзӣҳ
    • жӯӨж—¶redologеӨ„дәҺprepareзҠ¶жҖҒпјҲprepareж ҮзӯҫпјүпјҢз„¶еҗҺе‘ҠиҜүжү§иЎҢеҷЁжү§иЎҢе®ҢжҲҗпјҢйҡҸж—¶еҸҜд»ҘжҸҗдәӨдәӢеҠЎ
      • еҜ№е…¶д»–дәӢеҠЎжқҘиҜҙпјҢеҲҡеҲҡдҝ®ж”№зҡ„еҶ…еӯҳжҳҜдёҚеҸҜи§Ғзҡ„
  4. жү§иЎҢеҷЁз”ҹжҲҗиҝҷдёӘж“ҚдҪңзҡ„binlogпјҲйҖ»иҫ‘и®°еҪ•пјүе№¶еҶҷе…ҘзЈҒзӣҳ
    • binlogеҶҷжҲҗеҠҹдәӢеҠЎе°ұз®—жҲҗеҠҹпјҢеҸҜд»ҘжҸҗдәӨдәӢеҠЎ
      • е“ӘжҖ•еҙ©жәғжҒўеӨҚпјҢд№ҹдјҡжҒўеӨҚbinlogеҶҷжҲҗеҠҹзҡ„дәӢеҠЎпјҲжӯӨж—¶еҜ№еә”зҡ„redologеӨ„дәҺprepareзҠ¶жҖҒпјү
    • binlogеҰӮжһңжІЎеҶҷжҲҗеҠҹе°ұеӣһж»ҡпјҢеӣһж»ҡдјҡеҶҷredologпјҢжү“дёҠrollbackж ҮзӯҫпјҢbinlogеҲҷдјҡзӣҙжҺҘдёўејғ
      • еҰӮжһңbinlogдёҚдёўејғпјҢеҲҷдјҡдј ж’ӯеҲ°д»Һеә“
  5. жү§иЎҢеҷЁи°ғз”ЁInnoDBзҡ„жҸҗдәӨдәӢеҠЎжҺҘеҸЈпјҢInnoDBжҠҠеҲҡеҲҡеҶҷе…Ҙзҡ„redologж”№жҲҗcommitзҠ¶жҖҒпјҢжӣҙж–°е®ҢжҲҗ
    • redologжү“дёҠдәҶcommitж Үзӯҫ
    • commitиЎЁзӨәдёӨдёӘж—Ҙеҝ—йғҪз”ҹж•ҲдәҶ
    • commitе®ҢжҲҗеҗҺжүҚдјҡиҝ”еӣһе®ўжҲ·з«Ҝ

sync_binlog

redologзҡ„дёӨйҳ¶ж®өжҸҗдәӨ

  1. зӣ®зҡ„пјҡдёәдәҶи®©redologе’Ңbinlogзҡ„йҖ»иҫ‘дёҖиҮҙ
  2. и„‘жҙһпјҡеҒҮи®ҫдёҚйҮҮз”ЁдёӨйҳ¶ж®өжҸҗдәӨ
    • жңүдёӨз§ҚйЎәеәҸпјҡredolog->binlogпјҢжҲ–иҖ…binlog->redolog
    • жӯӨж—¶еҸҜд»Ҙи®Өдёәredologе’Ңbinlogе®Ңе…ЁзӢ¬з«Ӣ
      • еҙ©жәғжҒўеӨҚе®Ңе…Ёдҫқиө–дәҺredologпјҲеҺҹеә“пјүпјҢжҒўеӨҚдёҙж—¶еә“е®Ңе…Ёдҫқиө–дәҺbinlog
    • жҢүз…§дёҠйқўзҡ„дёӨз§ҚйЎәеәҸпјҢйғҪдјҡеҜјиҮҙredologе’ҢbinlogйҖ»иҫ‘дёҠзҡ„дёҚдёҖиҮҙ
      • еҒҮи®ҫеҺҹеә“crashеҗҺжү§иЎҢеҺҹеә“жҒўеӨҚ+жү§иЎҢдёҙж—¶еә“жҒўеӨҚпјҢжҒўеӨҚеҮәжқҘзҡ„ж•°жҚ®жҳҜдёҚдёҖиҮҙзҡ„пјҲдё»д»ҺдёҚдёҖиҮҙпјү
  3. жҒўеӨҚжё…йҶ’пјҡдёӨйҳ¶ж®өжҸҗдәӨпјҲеҒҮи®ҫжҳҜеҸҢ1зҡ„й…ҚзҪ®пјү
    • redolog prepare + binlogжҲҗеҠҹпјҢжҸҗдәӨдәӢеҠЎпјҢеҙ©жәғжҒўеӨҚеҗҺд№ҹдјҡз»§з»ӯжҸҗдәӨдәӢеҠЎпјҲredolog commitпјүпјҢйҖ»иҫ‘дёҖиҮҙ
    • redolog prepare + binlogеӨұиҙҘпјҢеӣһж»ҡдәӢеҠЎпјҢеҙ©жәғжҒўеӨҚеҗҺд№ҹдјҡз»§з»ӯеӣһж»ҡдәӢеҠЎпјҲredolog rollbackпјүпјҢйҖ»иҫ‘дёҖиҮҙ
      • дёҖдёӘдәӢеҠЎзҡ„binlogжҳҜжңүеӣәе®ҡж јејҸзҡ„
      • redologдёҺbinlogжҳҜйҖҡиҝҮдәӢеҠЎidпјҲXIDпјүиҝӣиЎҢе…іиҒ”зҡ„
      • жӯӨж—¶binglogдёӯжІЎжңүеҜ№еә”зҡ„и®°еҪ•пјҢдәӢеҠЎи®°еҪ•жҳҜдёҚе®Ңж•ҙзҡ„
    • еҙ©жәғжҒўеӨҚеҗҺжҳҜдјҡд»ҺcheckpointејҖе§ӢеҫҖеҗҺдё»еҠЁеҲ·ж•°жҚ®
  4. йҮҮз”ЁйқһеҸҢ1зҡ„й…ҚзҪ®пјҢеңЁжһҒз«ҜзҺҜеўғдёӢдјҡеҮәзҺ°redologдёҺbinlogдёҚдёҖиҮҙзҡ„жғ…еҶө
    • дјҳе…ҲдҝқиҜҒredologпјҢе…ҲеҺҹеә“еҙ©жәғжҒўеӨҚпјҢеҶҚеӨ„зҗҶд»Һеә“пјҲеҺҹеә“еҸҜд»ҘеңЁеҙ©жәғжҒўеӨҚеҗҺпјҢйҮҚж–°еҒҡдёҖж¬Ўе…ЁйҮҸеӨҮд»ҪпјҢйҮҚе»әд»Һеә“пјү

redolog->binlog

  1. еҸҚиҜҒпјҡдәӢеҠЎзҡ„жҢҒд№…жҖ§й—®йўҳ
  2. еҜ№InnoDBиҖҢиЁҖпјҢredologжңүcommitж ҮзӯҫпјҢд»ЈиЎЁиҝҷдёӘдәӢеҠЎдёҚиғҪеҶҚеӣһж»ҡ
  3. еҒҮиӢҘеҗҺз»ӯзҡ„binlogеҶҷеӨұиҙҘдәҶпјҢжӯӨж—¶ж•°жҚ®дёҺbinlogжҳҜдёҚдёҖиҮҙзҡ„пјҢдё»д»Һд№ҹдёҚдёҖиҮҙ

еҸӘжңүbinlog

binlogжҳҜйҖ»иҫ‘ж—Ҙеҝ—пјҢжІЎжңүи®°еҪ•ж•°жҚ®йЎөзҡ„жӣҙж–°з»ҶиҠӮпјҢ_жІЎжңүиғҪеҠӣжҒўеӨҚж•°жҚ®йЎө_

еҸӘжңүredolog

  1. еҰӮжһңд»…д»Һеҙ©жәғжҒўеӨҚзҡ„и§’еәҰжқҘиҜҙпјҢbinlogжҳҜеҸҜд»Ҙе…іжҺүзҡ„
    • еӣ дёәеҙ©жәғжҒўеӨҚжҳҜredologзҡ„еҠҹиғҪ
    • жӯӨж—¶жІЎжңүдёӨйҳ¶ж®өжҸҗдәӨпјҢдҪҶзі»з»ҹдҫқз„¶жҳҜcrash-safeзҡ„
  2. дҪҶзәҝдёҠдёҖиҲ¬йғҪдјҡжү“ејҖbinlogпјҢbinlogжңүзқҖredologж— жі•жӣҝд»Јзҡ„еҠҹиғҪ
    • йҰ–е…ҲжҳҜеҪ’жЎЈеҠҹиғҪпјҢredologжҳҜеҫӘзҺҜеҶҷпјҢиө·дёҚеҲ°еҪ’жЎЈзҡ„дҪңз”Ё
    • binlogеӨҚеҲ¶пјҡMySQLй«ҳеҸҜз”Ёзҡ„еҹәзЎҖ
      • дё»д»ҺеӨҚеҲ¶
      • ејӮжһ„зі»з»ҹпјҲеҰӮж•°жҚ®еҲҶжһҗзі»з»ҹпјүдјҡж¶Ҳиҙ№MySQLзҡ„binlogжқҘжӣҙж–°иҮӘе·ұзҡ„ж•°жҚ®

еҙ©жәғжҒўеӨҚ

еҲӨж–ӯйҖ»иҫ‘

  1. еҰӮжһңredologйҮҢйқўзҡ„дәӢеҠЎжҳҜе®Ңж•ҙзҡ„пјҢеҚіе·Із»ҸжңүдәҶcommitж ҮзӯҫпјҢйӮЈд№ҲзӣҙжҺҘжҸҗдәӨ
  2. еҰӮжһңredologйҮҢйқўзҡ„дәӢеҠЎеҸӘжңүprepareж ҮзӯҫпјҢеҲҷйңҖиҰҒеҲӨж–ӯдәӢеҠЎеҜ№еә”зҡ„binlogжҳҜеҗҰеӯҳеңЁе№¶е®Ңж•ҙ
    • a. еҰӮжһңжҳҜпјҢеҲҷжҸҗдәӨдәӢеҠЎ
    • b. еҰӮжһңеҗҰпјҢеҲҷеӣһж»ҡдәӢеҠЎ

redologдёҺbinlogе…іиҒ”

  1. redologе’ҢbinlogжңүдёҖдёӘе…ұеҗҢзҡ„ж•°жҚ®еӯ—ж®өпјҡXID
  2. еҙ©жәғжҒўеӨҚж—¶пјҢдјҡжҢүйЎәеәҸжү«жҸҸredolog
    • еҰӮжһңзў°еҲ°ж—ўжңүprepareж ҮзӯҫеҸҲжңүcommitж Үзӯҫзҡ„redologпјҢе°ұзӣҙжҺҘжҸҗдәӨ
    • еҰӮжһңзў°еҲ°еҸӘжңүprepareж ҮзӯҫдҪҶжІЎжңүcommitж Үзӯҫзҡ„redologпјҢе°ұжӢҝзқҖеҜ№еә”зҡ„XIDеҺ»binlogжүҫеҜ№еә”зҡ„дәӢеҠЎ
      • еҗҺз»ӯйңҖиҰҒж ЎйӘҢдәӢеҠЎеҜ№еә”зҡ„binlogзҡ„е®Ңж•ҙжҖ§

binlogзҡ„е®Ңж•ҙжҖ§

  1. binlogж јејҸ
    • statementж јејҸпјҢжңҖеҗҺдјҡжңүCOMMMIT;
    • rowж јејҸпјҢжңҖеҗҺдјҡжңүдёҖдёӘXID Event
  2. д»ҺMySQL 5.6.2ејҖе§ӢпјҢеј•е…ҘдәҶbinlog-checksumпјҢз”ЁдәҺйӘҢиҜҒbinlogеҶ…е®№зҡ„жӯЈзЎ®жҖ§
    • binlogеҸҜиғҪз”ұдәҺзЈҒзӣҳеҺҹеӣ пјҢеңЁж—Ҙеҝ—дёӯй—ҙеҮәй”ҷпјҢMySQLеҸҜд»ҘйҖҡиҝҮж ЎйӘҢchecksumжқҘеҸ‘зҺ°

ејӮеёёйҮҚеҗҜ

ж—¶еҲ»A

  1. еҜ№еә”дёҠйқў2.bзҡ„жғ…еҶө
  2. redologиҝҳжңӘжҸҗдәӨпјҢbinlogиҝҳжңӘеҶҷпјҢеңЁеҙ©жәғжҒўеӨҚж—¶пјҢдәӢеҠЎдјҡеӣһж»ҡ
  3. з”ұдәҺbinlogиҝҳжІЎеҶҷпјҢд№ҹдёҚдјҡдј еҲ°еӨҮеә“

ж—¶еҲ»B

  1. еҜ№еә”дёҠйқўзҡ„2.aзҡ„жғ…еҶөпјҢеҙ©жәғжҒўеӨҚиҝҮзЁӢдёӯдәӢеҠЎдјҡиў«жҸҗдәӨ
  2. жӯӨж—¶binlogе·Із»ҸеҶҷе…ҘдәҶпјҢд№ӢеҗҺдјҡиў«д»Һеә“жҲ–дёҙж—¶еә“дҪҝз”Ё
    • еӣ жӯӨдё»еә“д№ҹйңҖиҰҒжҸҗдәӨиҝҷдёӘдәӢеҠЎпјҢдҝқиҜҒдё»д»ҺдёҖиҮҙ
