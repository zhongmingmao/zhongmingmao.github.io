  2. InnoDBй»ҳи®ӨдәӢеҠЎйҡ”зҰ»зә§еҲ«жҳҜRRпјҢйҖҡиҝҮMVCCе®һзҺ°

    • дәҢзә§зҙўеј•ж ‘пјҡеҸ¶еӯҗиҠӮзӮ№жҳҜдё»й”®еҖј
  2. дәҢзә§зҙўеј•ж ‘еҚ з”Ёзҡ„з©әй—ҙжҜ”иҒҡз°Үзҙўеј•ж ‘е°ҸеҫҲеӨҡ
  3. дјҳеҢ–еҷЁдјҡеңЁдҝқиҜҒйҖ»иҫ‘жӯЈзЎ®зҡ„еүҚжҸҗдёӢпјҢйҒҚеҺҶжңҖе°Ҹзҡ„зҙўеј•ж ‘пјҢе°ҪйҮҸеҮҸе°‘жү«жҸҸзҡ„ж•°жҚ®йҮҸ
    • й’ҲеҜ№ж— иҝҮж»ӨжқЎд»¶зҡ„countж“ҚдҪңпјҢж— и®әйҒҚеҺҶе“ӘдёҖйў—зҙўеј•ж ‘пјҢж•ҲжһңйғҪжҳҜдёҖж ·зҡ„
    • дјҳеҢ–еҷЁдјҡдёәcount(*)йҖүжӢ©жңҖдјҳзҡ„зҙўеј•ж ‘

countзҡ„жҖ§иғҪ

иҜӯд№ү

  1. count()жҳҜдёҖдёӘиҒҡеҗҲеҮҪж•°пјҢеҜ№дәҺиҝ”еӣһзҡ„з»“жһңйӣҶпјҢдёҖиЎҢдёҖиЎҢең°иҝӣиЎҢеҲӨж–ӯ
    • еҰӮжһңcountеҮҪж•°зҡ„еҸӮж•°еҖјдёҚжҳҜNULLпјҢзҙҜи®ЎеҖј+1пјҢеҗҰеҲҷдёҚеҠ пјҢжңҖеҗҺиҝ”еӣһзҙҜи®ЎеҖј
  2. count(еӯ—ж®өF)
    • еӯ—ж®өFжңүеҸҜиғҪдёәNULL
    • иЎЁзӨәиҝ”еӣһж»Ўи¶іжқЎд»¶зҡ„з»“жһңйӣҶйҮҢеӯ—ж®өFдёҚдёәNULLзҡ„жҖ»ж•°
  3. count(дё»й”®ID)гҖҒcount(1)гҖҒcount(*)
    • дёҚеҸҜиғҪдёәNULL
    • иЎЁзӨәиҝ”еӣһж»Ўи¶іжқЎд»¶зҡ„з»“жһңйӣҶзҡ„жҖ»ж•°
  4. ServerеұӮиҰҒд»Җд№Ҳеӯ—ж®өпјҢInnoDBеј•ж“Һе°ұиҝ”еӣһд»Җд№Ҳеӯ—ж®ө
    • count(*)дҫӢеӨ–пјҢ_дёҚиҝ”еӣһж•ҙиЎҢ_пјҢеҸӘиҝ”еӣһз©әиЎҢ

жҖ§иғҪеҜ№жҜ”

count(еӯ—ж®өF)

  1. еҰӮжһңеӯ—ж®өFе®ҡд№үдёәдёҚе…Ғи®ёдёәNULLпјҢдёҖиЎҢиЎҢең°д»Һи®°еҪ•йҮҢиҜ»еҮәиҝҷдёӘеӯ—ж®өпјҢеҲӨж–ӯйҖҡиҝҮеҗҺжҢүиЎҢзҙҜеҠ 
    • йҖҡиҝҮиЎЁз»“жһ„еҲӨж–ӯиҜҘеӯ—ж®өжҳҜдёҚеҸҜиғҪдёәNULL
  2. еҰӮжһңеӯ—ж®өFе®ҡд№үдёәе…Ғи®ёNULLпјҢдёҖиЎҢиЎҢең°д»Һи®°еҪ•йҮҢиҜ»еҮәиҝҷдёӘеӯ—ж®өпјҢеҲӨж–ӯйҖҡиҝҮеҗҺжҢүиЎҢзҙҜеҠ 
    • йҖҡиҝҮиЎЁз»“жһ„еҲӨж–ӯиҜҘеӯ—ж®өжҳҜжңүеҸҜиғҪдёәNULL
    • еҲӨж–ӯиҜҘеӯ—ж®өеҖјжҳҜеҗҰе®һйҷ…дёәNULL
  3. еҰӮжһңеӯ—ж®өFдёҠжІЎжңүдәҢзә§зҙўеј•пјҢеҸӘиғҪйҒҚеҺҶж•ҙеј иЎЁпјҲиҒҡз°Үзҙўеј•пјү
  4. з”ұдәҺInnoDBеҝ…йЎ»иҝ”еӣһеӯ—ж®өFпјҢеӣ жӯӨдјҳеҢ–еҷЁиғҪеҒҡеҮәзҡ„дјҳеҢ–еҶізӯ–е°ҶеҮҸе°‘
    • дҫӢеҰӮдёҚиғҪйҖүжӢ©жңҖдјҳзҡ„зҙўеј•жқҘйҒҚеҺҶ

count(дё»й”®ID)

  1. InnoDBдјҡйҒҚеҺҶж•ҙеј иЎЁпјҲиҒҡз°Үзҙўеј•пјүпјҢжҠҠжҜҸдёҖиЎҢзҡ„idеҖјеҸ–еҮәжқҘпјҢиҝ”еӣһз»ҷServerеұӮ
  2. ServerеұӮжӢҝеҲ°idеҗҺпјҢеҲӨж–ӯдёәдёҚеҸҜиғҪдёәNULLпјҢз„¶еҗҺжҢүиЎҢзҙҜеҠ 
  3. дјҳеҢ–еҷЁеҸҜиғҪдјҡйҖүжӢ©жңҖдјҳзҡ„зҙўеј•жқҘйҒҚеҺҶ

count(1)

  1. InnoDBеј•ж“ҺдјҡйҒҚеҺҶж•ҙеј иЎЁпјҲиҒҡз°Үзҙўеј•пјүпјҢдҪҶдёҚеҸ–еҖј
  2. ServerеұӮеҜ№дәҺиҝ”еӣһзҡ„жҜҸдёҖиЎҢпјҢж”ҫдёҖдёӘж•°еӯ—1иҝӣеҺ»пјҢеҲӨж–ӯжҳҜдёҚеҸҜиғҪдёәNULLпјҢжҢүиЎҢзҙҜеҠ 
  3. count(1)жҜ”count(дё»й”®ID)еҝ«пјҢеӣ дёәcount(дё»й”®ID)дјҡж¶үеҸҠеҲ°дёӨйғЁеҲҶж“ҚдҪң
    • и§Јжһҗж•°жҚ®иЎҢ
    • жӢ·иҙқеӯ—ж®өеҖј

count(*)

  1. count(*)дёҚдјҡжҠҠжүҖжңүеҖјйғҪеҸ–еҮәжқҘпјҢиҖҢжҳҜдё“й—ЁеҒҡдәҶдјҳеҢ–пјҢдёҚеҸ–еҖјпјҢеӣ дёәгҖҺ*гҖҸиӮҜе®ҡдёҚдёәNULLпјҢжҢүиЎҢзҙҜеҠ 
  2. дёҚеҸ–еҖјпјҡInnoDBиҝ”еӣһдёҖдёӘз©әиЎҢпјҢе‘ҠиҜүServerеұӮдёҚжҳҜNULLпјҢеҸҜд»Ҙи®Ўж•°

ж•ҲзҺҮжҺ’еәҸ

  1. count(еӯ—ж®өF) < count(дё»й”®ID) < count(1) вүҲ count(*)
  2. е°ҪйҮҸдҪҝз”Ёcount(*)

ж ·дҫӢ

user_idsдёҠж— зҙўеј•пјҢиҖҢInnoDBеҸҲеҝ…йЎ»иҝ”еӣһuser_idsеӯ—ж®өпјҢеҸӘиғҪйҒҚеҺҶиҒҡз°Үзҙўеј•

жңүзҙўеј•
  1. serial_idдёҠжңүзҙўеј•пјҢеҸҜд»ҘйҒҚеҺҶuniq_serial_id_source_index
  2. дҪҶз”ұдәҺInnoDBеҝ…йЎ»иҝ”еӣһserial_idеӯ—ж®өпјҢеӣ жӯӨдёҚдјҡйҒҚеҺҶйҖ»иҫ‘з»“жһңзӯүд»·зҡ„жӣҙдјҳйҖүжӢ©idx_create_time
    • еҰӮжһңйҖүжӢ©idx_create_timeпјҢе№¶дё”иҝ”еӣһserial_idеӯ—ж®өпјҢиҝҷж„Ҹе‘ізқҖеҝ…йЎ»еӣһиЎЁ
