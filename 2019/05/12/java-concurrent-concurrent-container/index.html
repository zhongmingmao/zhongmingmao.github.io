Javaе№¶еҸ‘ -- е№¶еҸ‘е®№еҷЁ

Created2019-05-12|JavaConcurrent
|Word Count:1.3k|Reading Time:4mins

еҗҢжӯҘе®№еҷЁ

  1. Java 1.5д№ӢеүҚжҸҗдҫӣзҡ„еҗҢжӯҘе®№еҷЁиҷҪз„¶д№ҹиғҪдҝқиҜҒзәҝзЁӢе®үе…ЁпјҢдҪҶжҖ§иғҪеҫҲе·®
  2. Javaдёӯзҡ„е®№еҷЁдё»иҰҒеҲҶдёәеӣӣеӨ§зұ»пјҢеҲҶеҲ«дёәListгҖҒMapгҖҒSetе’ҢQueueпјҢе№¶дёҚжҳҜжүҖжңүзҡ„Javaе®№еҷЁйғҪжҳҜзәҝзЁӢе®үе…Ёзҡ„
  3. е°ҶйқһзәҝзЁӢе®үе…Ёзҡ„е®№еҷЁеҸҳжҲҗзәҝзЁӢе®үе…Ёзҡ„е®№еҷЁзҡ„з®ҖеҚ•ж–№жЎҲпјҡsynchronized
    • жҠҠйқһзәҝзЁӢе®үе…Ёзҡ„е®№еҷЁе°ҒиЈ…еңЁеҜ№иұЎеҶ…йғЁпјҢз„¶еҗҺжҺ§еҲ¶еҘҪи®ҝй—®и·Ҝеҫ„еҚіеҸҜ

зәҝзЁӢе®үе…Ёзҡ„ArrayList

Collections.synchronized

з»„еҗҲж“ҚдҪңеӯҳеңЁз«ҹжҖҒжқЎд»¶й—®йўҳ

  1. дёҠйқўзҡ„addIfNotExistе°ұеҢ…еҗ«з»„еҗҲж“ҚдҪң
  2. з»„еҗҲж“ҚдҪңеҫҖеҫҖйҡҗи—ҸзқҖз«ҹжҖҒжқЎд»¶й—®йўҳпјҢеҚідҫҝжҜҸдёӘж“ҚдҪңйғҪиғҪдҝқиҜҒеҺҹеӯҗжҖ§пјҢд№ҹдёҚиғҪдҝқиҜҒз»„еҗҲж“ҚдҪңзҡ„еҺҹеӯҗжҖ§
  3. з”Ёиҝӯд»ЈеҷЁйҒҚеҺҶеҗҢжӯҘе®№еҷЁд№ҹеӯҳеңЁз«ҹжҖҒжқЎд»¶й—®йўҳпјҢеӣ дёә_з»„еҗҲж“ҚдҪңдёҚе…·еӨҮеҺҹеӯҗжҖ§_

е№¶еҸ‘е®№еҷЁ

  1. JavaеңЁ1.5д№ӢеүҚжүҖи°“зҡ„зәҝзЁӢе®үе…Ёе®№еҷЁпјҢдё»иҰҒжҢҮзҡ„жҳҜеҗҢжӯҘе®№еҷЁ
  2. еҗҢжӯҘе®№еҷЁжңҖеӨ§зҡ„й—®йўҳжҳҜжҖ§иғҪе·®пјҢжүҖжңүж–№жі•йғҪз”ЁsynchronizedжқҘдҝқиҜҒдә’ж–ҘпјҢдёІиЎҢеәҰеӨӘй«ҳ
  3. еңЁJava 1.5жҸҗдҫӣдәҶжҖ§иғҪжӣҙй«ҳзҡ„е®№еҷЁпјҢз§°дёәе№¶еҸ‘е®№еҷЁ

еҲҶзұ»

е№¶еҸ‘е®№еҷЁж•°йҮҸдј—еӨҡпјҢдҪҶдҫқж—§еҸҜд»ҘеҲҶжҲҗеӣӣеӨ§зұ»пјҡListгҖҒMapгҖҒSetе’ҢQueue

List

  1. ListйҮҢйқўеҸӘжңүдёҖдёӘе®һзҺ°зұ»е°ұжҳҜCopyOnWriteArrayList
  2. CopyOnWriteеҚіеңЁжү§иЎҢеҶҷж“ҚдҪңзҡ„ж—¶еҖҷдјҡе°Ҷе…ұдә«еҸҳйҮҸйҮҚж–°еӨҚеҲ¶дёҖд»ҪеҮәжқҘпјҢиҝҷж ·зҡ„еҘҪеӨ„жҳҜиҜ»ж“ҚдҪңжҳҜе®Ңе…Ёж— й”Ғзҡ„
  3. CopyOnWriteArrayListеҶ…йғЁз»ҙжҠӨдёҖдёӘж•°з»„пјҢжҲҗе‘ҳеҸҳйҮҸarrayжҢҮеҗ‘иҝҷдёӘеҶ…йғЁж•°з»„пјҢжүҖжңүзҡ„иҜ»ж“ҚдҪңйғҪжҳҜеҹәдәҺarrayиҝӣиЎҢзҡ„
  4. еҰӮжһңеңЁйҒҚеҺҶarrayзҡ„еҗҢж—¶пјҢиҝҳжңүдёҖдёӘеҶҷж“ҚдҪң
    • дјҡе°ҶarrayеӨҚеҲ¶дёҖд»ҪпјҢз„¶еҗҺеңЁж–°еӨҚеҲ¶зҡ„ж•°з»„дёҠжү§иЎҢеҶҷж“ҚдҪңпјҢжү§иЎҢе®Ңд№ӢеҗҺеҶҚе°ҶarrayжҢҮеҗ‘иҝҷдёӘж–°зҡ„ж•°з»„
  5. еӣ жӯӨиҜ»еҶҷжҳҜе№¶иЎҢзҡ„пјҢ йҒҚеҺҶж“ҚдҪңдёҖзӣҙйғҪжҳҜеҹәдәҺеҺҹarrayжү§иЎҢзҡ„пјҢиҖҢеҶҷж“ҚдҪңеҲҷжҳҜеҹәдәҺж–°arrayжү§иЎҢзҡ„
  6. еә”з”ЁеңәжҷҜпјҡд»…йҖӮз”ЁдәҺеҶҷж“ҚдҪңйқһеёёе°‘зҡ„еңәжҷҜпјҢиҖҢдё”иғҪеӨҹе®№еҝҚиҜ»еҶҷзҡ„зҹӯжҡӮдёҚдёҖиҮҙ
  7. CopyOnWriteArrayListзҡ„иҝӯд»ЈеҷЁжҳҜеҸӘиҜ»зҡ„пјҢдёҚж”ҜжҢҒеўһеҲ ж”№пјҢеӣ дёәеҜ№еҝ«з…§иҝӣиЎҢеўһеҲ ж”№жҳҜжІЎжңүж„Ҹд№үзҡ„

Map

  1. MapжҺҘеҸЈзҡ„дёӨдёӘе®һзҺ°пјҡConcurrentHashMapе’ҢConcurrentSkipListMap
  2. ConcurrentHashMapзҡ„keyжҳҜж— еәҸзҡ„пјҢиҖҢConcurrentSkipListMapзҡ„keyжҳҜжңүеәҸзҡ„
  3. ConcurrentSkipListMapйҮҢйқўзҡ„SkipListжң¬иә«жҳҜдёҖз§Қж•°жҚ®з»“жһ„пјҢзҝ»иҜ‘жҲҗи·іиЎЁ
    • и·іиЎЁжү§иЎҢжҸ’е…ҘгҖҒеҲ йҷӨе’ҢжҹҘиҜўж“ҚдҪңзҡ„е№іеқҮеӨҚжқӮеәҰдёәO(log n)
    • зҗҶи®әдёҠдёҺе№¶еҸ‘зәҝзЁӢж•°ж— е…іпјҢйҖӮз”ЁдәҺе№¶еҸ‘еәҰйқһеёёй«ҳзҡ„жғ…еҶөпјҲConcurrentHashMapзҡ„жҖ§иғҪд№ҹдёҚиғҪж»Ўи¶іиҰҒжұӮпјү
йӣҶеҗҲзұ» Key Value зәҝзЁӢе®үе…Ё
HashMap е…Ғи®ёдёәnull е…Ғи®ёдёәnull еҗҰ
TreeMap дёҚе…Ғи®ёдёәnull е…Ғи®ёдёәnull еҗҰ
HashTable дёҚе…Ғи®ёдёәnull дёҚе…Ғи®ёдёәnull жҳҜ
ConcurrentHashMap дёҚе…Ғи®ёдёәnull дёҚе…Ғи®ёдёәnull жҳҜ
ConcurrentSkipListMap дёҚе…Ғи®ёдёәnull дёҚе…Ғи®ёдёәnull жҳҜ

Set

  1. SetжҺҘеҸЈзҡ„дёӨдёӘе®һзҺ°пјҡCopyOnWriteArraySetе’ҢConcurrentSkipListSet
  2. еҺҹзҗҶдёҺCopyOnWriteArrayListе’ҢConcurrentSkipListMapзұ»дјј

Queue

  1. JUCдёӯзҡ„Queueзұ»зҡ„е№¶еҸ‘е®№еҷЁжҳҜжңҖеӨҚжқӮзҡ„пјҢеҸҜд»Ҙд»ҺдёӨдёӘз»ҙеәҰеҲҶзұ»пјҢйҳ»еЎһ/йқһйҳ»еЎһгҖҒеҚ•з«Ҝ/еҸҢз«Ҝ
  2. йҳ»еЎһ/йқһйҳ»еЎһпјҡйҳ»еЎһжҢҮзҡ„жҳҜеҪ“йҳҹеҲ—е·Іж»Ўж—¶пјҢе…Ҙйҳҹж“ҚдҪңйҳ»еЎһпјӣеҪ“йҳҹеҲ—е·Із©әж—¶пјҢеҮәйҳҹж“ҚдҪңйҳ»еЎһ
  3. еҚ•з«Ҝ/еҸҢз«ҜпјҡеҚ•з«ҜжҢҮзҡ„жҳҜеҸӘиғҪйҳҹе°ҫе…ҘйҳҹпјҢйҳҹйҰ–еҮәйҳҹпјӣеҸҢз«ҜжҢҮзҡ„жҳҜйҳҹйҰ–йҳҹе°ҫзҡҶеҸҜеҮәйҳҹе…Ҙйҳҹ
  4. еңЁJUCдёӯпјҢйҳ»еЎһйҳҹеҲ—з”ЁBlockingе…ій”®еӯ—ж ҮиҜҶпјҢеҚ•з«ҜйҳҹеҲ—з”ЁQueueж ҮиҜҶпјҢеҸҢз«ҜйҳҹеҲ—з”ЁQequeж ҮиҜҶ

еҚ•з«Ҝйҳ»еЎһйҳҹеҲ—

  1. е…¶е®һзҺ°еҢ…жӢ¬
    • ArrayBlockingQueue
    • LinkedBlockingQueue
    • SynchronousQueue
    • LinkedTransferQueue
    • PriorityBlockingQueue
    • DelayQueue
  2. еҶ…йғЁдёҖиҲ¬йғҪдјҡжҢҒжңүдёҖдёӘйҳҹеҲ—
    • иҜҘйҳҹеҲ—еҸҜд»ҘжҳҜж•°з»„пјҲArrayBlockingQueueпјү
    • д№ҹеҸҜд»ҘжҳҜй“ҫиЎЁпјҲLinkedBlockingQueueпјү
    • з”ҡиҮідёҚжҢҒжңүйҳҹеҲ—пјҲSynchronousQueueпјүпјҢ_з”ҹдә§иҖ…зәҝзЁӢзҡ„е…Ҙйҳҹж“ҚдҪңеҝ…йЎ»зӯүеҫ…ж¶Ҳиҙ№иҖ…зәҝзЁӢйғҪеҮәйҳҹж“ҚдҪң_
  3. LinkedTransferQueueиһҚеҗҲдәҶLinkedBlockingQueueе’ҢSynchronousQueueзҡ„еҠҹиғҪпјҢжҖ§иғҪжҜ”LinkedBlockingQueueжӣҙеҘҪ
  4. PriorityBlockingQueueж”ҜжҢҒжҢүдјҳе…Ҳзә§еҮәйҳҹ
  5. DelayQueueж”ҜжҢҒе»¶ж—¶йҳҹеҲ—

еҸҢз«Ҝйҳ»еЎһйҳҹеҲ—

е…¶е®һзҺ°жҳҜLinkedBlockingDeque

еҚ•з«Ҝйқһйҳ»еЎһйҳҹеҲ—

е…¶е®һзҺ°жҳҜConcurrentLinkedQueue

еҸҢз«Ҝйқһйҳ»еЎһйҳҹеҲ—

е…¶е®һзҺ°жҳҜConcurrentLinkedDeque

жҳҜеҗҰжңүз•Ң

  1. дҪҝз”ЁйҳҹеҲ—ж—¶пјҢиҰҒж јеӨ–жіЁж„ҸйҳҹеҲ—жҳҜеҗҰж”ҜжҢҒжңүз•Ң
  2. е®һйҷ…е·ҘдҪңдёӯпјҢдёҖиҲ¬дёҚе»әи®®дҪҝз”Ёж— з•Ңзҡ„йҳҹеҲ—пјҢеӣ дёәжңүеҸҜиғҪдјҡеҜјиҮҙ_OOM_
  3. дёҠйқўжҸҗеҲ°зҡ„QueueпјҢеҸӘжңүArrayBlockingQueueе’ҢLinkedBlockingQueueжҳҜж”ҜжҢҒжңүз•Ңзҡ„
