йў„жңҹеҠҹиғҪ

  1. жңүдёҡеҠЎ Code з Ғж ҮиҜҶ - HTTP Code з Ғжңүйҷҗ
  2. е®үе…Ё - еҜ№еҶ…еҜ№еӨ–еұ•зӨәдёҚеҗҢзҡ„й”ҷиҜҜдҝЎжҒҜ

и®ҫи®Ўж–№ејҸ

  1. ж— и®әиҜ·жұӮжҲҗеҠҹдёҺеҗҰпјҢйғҪиҝ”еӣһ 200пјҢеңЁ HTTP Body дёӯеҢ…еҗ«й”ҷиҜҜдҝЎжҒҜ
    • HTTP Code йҖҡеёёд»ЈиЎЁ HTTP Transport еұӮзҡ„зҠ¶жҖҒдҝЎжҒҜ
    • зјәзӮ№пјҡжҖ§иғҪдёҚдҪіпјҲйңҖи§Јжһҗ Bodyпјү + еҜ№е®ўжҲ·з«ҜдёҚеҸӢеҘҪ
  2. иҝ”еӣһ 4xx or 5xxпјҢе№¶еңЁ Body дёӯиҝ”еӣһз®ҖеҚ•зҡ„й”ҷиҜҜдҝЎжҒҜ
  3. иҝ”еӣһ 4xx or 5xxпјҢе№¶еңЁ Body дёӯиҝ”еӣһиҜҰз»Ҷзҡ„й”ҷиҜҜдҝЎжҒҜ - жҺЁиҚҗ

и®ҫи®ЎжҖқи·Ҝ

  1. еҢәеҲ«дәҺ http status codeпјҢдёҡеҠЎз ҒйңҖиҰҒжңүдёҖе®ҡзҡ„и§„еҲҷпјҢеҸҜд»ҘйҖҡиҝҮдёҡеҠЎз ҒеҲӨж–ӯеҮәжҳҜе“Әзұ»й”ҷиҜҜ
  2. иҜ·жұӮеҮәй”ҷж—¶пјҢеҸҜд»ҘйҖҡиҝҮ http status codeпјҢзӣҙжҺҘж„ҹзҹҘеҲ°иҜ·жұӮеҮәй”ҷ
  3. йңҖиҰҒеңЁиҜ·жұӮеҮәй”ҷж—¶пјҢиҝ”еӣһиҜҰз»Ҷзҡ„дҝЎжҒҜпјҡCode + Message + ReferenceпјҲOptionalпјү
  4. иҝ”еӣһзҡ„й”ҷиҜҜдҝЎжҒҜпјҢйңҖиҰҒжҳҜеҸҜд»ҘзӣҙжҺҘеұ•зӨәз»ҷз”ЁжҲ·зҡ„е®үе…ЁдҝЎжҒҜпјҢдёҚиғҪеҢ…еҗ«ж•Ҹж„ҹдҝЎжҒҜ
    • дҪҶеҗҢж—¶д№ҹиҰҒжңүеҶ…йғЁжӣҙиҜҰз»Ҷзҡ„й”ҷиҜҜдҝЎжҒҜпјҢж–№дҫҝ Debug
  5. иҝ”еӣһзҡ„ж•°жҚ®ж јејҸеә”иҜҘжҳҜеӣәе®ҡзҡ„пјҢи§„иҢғзҡ„
  6. й”ҷиҜҜдҝЎжҒҜиҰҒдҝқжҢҒз®ҖжҙҒпјҢе№¶жҸҗдҫӣжңүз”Ёзҡ„дҝЎжҒҜ

Biz Code

зәҜж•°еӯ—иЎЁзӨәпјҢдёҚеҗҢйғЁдҪҚд»ЈиЎЁдёҚеҗҢзҡ„жңҚеҠЎпјҢдёҚеҗҢзҡ„жЁЎеқ—

HTTP Code

net/http - 60

Code Short Desc
1xx жҢҮзӨәдҝЎжҒҜ иЎЁзӨәиҜ·жұӮе·ІжҺҘж”¶пјҢз»§з»ӯеӨ„зҗҶ
2xx иҜ·жұӮжҲҗеҠҹ иЎЁзӨәжҲҗеҠҹеӨ„зҗҶдәҶиҜ·жұӮ
3xx иҜ·жұӮиў«йҮҚе®ҡеҗ‘ иЎЁзӨәиҰҒе®ҢжҲҗиҜ·жұӮпјҢйңҖиҰҒиҝӣдёҖжӯҘж“ҚдҪң
4xx иҜ·жұӮй”ҷиҜҜ д»ЈиЎЁиҜ·жұӮеҸҜиғҪеҮәй”ҷпјҢйҖҡеёёжҳҜе®ўжҲ·з«ҜеҮәй”ҷ
5xx жңҚеҠЎз«Ҝй”ҷиҜҜ жңҚеҠЎеҷЁеңЁе°қиҜ•еӨ„зҗҶиҜ·жұӮж—¶еҸ‘з”ҹй”ҷиҜҜ

дёҚе»әи®® http status code еӨӘеӨҡ

  1. 200 - иЎЁзӨәиҜ·жұӮжҲҗеҠҹжү§иЎҢ
  2. 400 - иЎЁзӨәе®ўжҲ·з«ҜеҮәй”ҷ
  3. 500 - иЎЁзӨәжңҚеҠЎз«ҜеҮәй”ҷ

жңҖеӨҡеҶҚеҠ  3 дёӘ Code

  1. 401 - иЎЁзӨәи®ӨиҜҒеӨұиҙҘ
  2. 403 - иЎЁзӨәжҺҲжқғеӨұиҙҘ
  3. 404 - иЎЁзӨәиө„жәҗжүҫдёҚеҲ°

Error Message

  1. еҜ№еӨ–жҡҙйңІзҡ„й”ҷиҜҜпјҢз»ҹдёҖеӨ§еҶҷејҖеӨҙпјҢз»“е°ҫдёҚйңҖиҰҒ .
  2. еҜ№еӨ–жҡҙйңІзҡ„й”ҷиҜҜиҰҒз®ҖжҙҒпјҢе№¶иғҪеҮҶзЎ®иҜҙжҳҺй—®йўҳ
  3. еҜ№еӨ–жҡҙйңІзҡ„й”ҷиҜҜиҜҙжҳҺпјҢеә”иҜҘжҳҜжҖҺд№ҲеҒҡпјҢиҖҢдёҚжҳҜе“ӘйҮҢй”ҷ

еҰӮжһңиҝ”еӣһз»“жһңдёӯеӯҳеңЁ code еӯ—ж®өпјҢиЎЁзӨәи°ғз”Ё API жҺҘеҸЈеӨұиҙҘ
