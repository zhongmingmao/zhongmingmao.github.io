Kubernetes - Daemonset

Created2022-10-26|Cloud NativeCloud Native FoundationKubernetes
|Word Count:763|Reading Time:4mins

YAML

与 Deployment 非常类似，但缺少 replicas，因为 Daemonset 的意图是在每个 Node 上部署一个 Pod

使用

Taint + Toleration

Node 的 Taint，Pod 的 Toleration

移除 Master 的 Taint

Master 再次添加 Taint，不会驱逐 Pod

为 Pod 添加 Toleration

Static Pod

Static Pod 不受 Kubernetes 管控，不与 apiserver、scheduler 发生关系，只受 kubelet 管控

kubelet 会定期检查 staticPodPath 中的文件，发现变化时会调用 CRI 创建或者删除 Static Pod
