Kubernetes - Deployment

Created2022-10-25|Cloud NativeCloud Native FoundationKubernetes
|Word count:1.8k|Reading time:11min

API

YAML

  1. replicas
    • 副本数
  2. selector
    • 筛选出要被 Deployment 管理的 Pod 对象
    • matchLabels
      • 定义了 Pod 对象应该携带的 Label，必须和 template.labels 完全相同
      • 否则 Deployment 就会找不到要控制的 Pod 对象，kube-apiserver 会告知 YAML 格式校验错误，无法创建

Job 中的 Pod 一般不会被其它对象使用；而 Deployment 中的 Pod 有可能会被其它对象引用来管理，例如 Service

Deployment 和 Pod 是一种松散的组织关系，Deployment 并不实际持有 Pod，通过 Label 实现松耦合
