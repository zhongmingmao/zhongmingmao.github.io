GOPATH

  1. Go йҰ–ж¬ЎејҖжәҗж—¶пјҢеҶ…зҪ®дәҶ GOPATH зҡ„жһ„е»әжЁЎејҸ
  2. Go зј–иҜ‘еҷЁеҸҜд»ҘеңЁжң¬ең° GOPATH дёӢжҗңзҙў Go зЁӢеәҸдҫқиө–зҡ„з¬¬дёүж–№еҢ…
    • еҰӮжһңеӯҳеңЁпјҢеҲҷдҪҝз”ЁиҝҷдёӘжң¬ең°еҢ…иҝӣиЎҢзј–иҜ‘пјӣеҰӮжһңдёҚеӯҳеңЁпјҢеҲҷдјҡжҠҘзј–иҜ‘й”ҷиҜҜ

ж— жі•жүҫеҲ°дҫқиө–еҢ…иҖҢжһ„е»әеӨұиҙҘ

жҗңзҙўи§„еҲҷ

  1. Go зЁӢеәҸйңҖиҰҒеҜје…Ҙ github.com/user/repo иҝҷдёӘеҢ…и·Ҝеҫ„пјҢеҒҮи®ҫ GOPATH дёә PathA:PathB
  2. еңЁ GOPATH жһ„е»әжЁЎејҸдёӢпјҢGo зј–иҜ‘еҷЁеңЁзј–иҜ‘ Go зЁӢеәҸж—¶пјҢдјҡд»ҺдёӢеҲ—и·Ҝеҫ„жҗңзҙўз¬¬дёүж–№еҢ…
    • PathA/src/github.com/user/repo
    • PathB/src/github.com/user/repo

GOPATH й»ҳи®ӨеҖјдёә $HOME/go

еҸҜд»ҘйҖҡиҝҮ go get е‘Ҫд»Өе°ҶзјәеӨұзҡ„з¬¬дёүж–№еҢ…дёӢиҪҪеҲ° GOPATH й…ҚзҪ®зҡ„зӣ®еҪ•дёӢпјҢе№¶зЎ®дҝқй—ҙжҺҘдҫқиө–д№ҹеӯҳеңЁ

go get дёӢиҪҪзҡ„еҢ…еҸӘжҳҜеҪ“еүҚж—¶еҲ»еҗ„дёӘдҫқиө–зҡ„жңҖж–°дё»зәҝзүҲжң¬пјҢж— жі•дҝқиҜҒ Reproducible Build
еҰӮжһңдҫқиө–еҢ…еј•е…ҘдәҶдёҚе…је®№зҡ„д»Јз ҒпјҢGo зЁӢеәҸе°Ҷзј–иҜ‘еӨұиҙҘ

  1. еңЁ GOPATH жһ„е»әжЁЎејҸдёӢпјҢGo зј–иҜ‘еҷЁе®һйҷ…дёҠе№¶жІЎжңүе…іжіЁ Go йЎ№зӣ®жүҖдҫқиө–зҡ„з¬¬дёүж–№еҢ…зҡ„зүҲжң¬
  2. Go ж ёеҝғејҖеҸ‘еӣўйҳҹеӣ жӯӨеј•е…ҘдәҶ vendor жңәеҲ¶

Vendor

  1. Go 1.5 еј•е…ҘдәҶ vendor жңәеҲ¶
  2. vendor жңәеҲ¶зҡ„жң¬иҙЁпјҡеңЁ Go йЎ№зӣ®зҡ„ vendor зӣ®еҪ•дёӢпјҢзј“еӯҳйЎ№зӣ®зҡ„жүҖжңүдҫқиө–еҢ…
  3. Go зј–иҜ‘еҷЁдјҡдјҳе…Ҳж„ҹзҹҘе’ҢдҪҝз”Ё vendor зӣ®еҪ•дёӢзј“еӯҳзҡ„з¬¬дёүж–№зүҲжң¬пјҢиҖҢдёҚжҳҜ GOPATH
  4. е°Ҷ vendor зӣ®еҪ•е’ҢйЎ№зӣ®жәҗз ҒдёҖе№¶жҸҗдәӨеҲ°д»Јз Ғд»“еә“еҗҺпјҢеҸҜд»Ҙе®һзҺ° Reproducible build

Go йЎ№зӣ®еҝ…йЎ»иҰҒдҪҚдәҺ GOPATH/src зҡ„зӣ®еҪ•дёӢпјҢжүҚиғҪејҖеҗҜ vendor жңәеҲ¶
еҰӮжһңдёҚж»Ўи¶іиҜҘи·Ҝеҫ„иҰҒжұӮпјҢGo зј–иҜ‘еҷЁдјҡеҝҪз•Ҙ Go йЎ№зӣ®дёӢзҡ„ vendor зӣ®еҪ•

vendor жңәеҲ¶иҷҪз„¶и§ЈеҶідәҶ Reproducible Build зҡ„й—®йўҳпјҢдҪҶејҖеҸ‘дҪ“йӘҢж¬ дҪі

  1. Go йЎ№зӣ®еҝ…йЎ»ж”ҫеңЁ GOPATH/src дёӢ
  2. еәһеӨ§зҡ„ vendor зӣ®еҪ•йңҖиҰҒжҸҗдәӨеҲ°д»Јз Ғд»“еә“
  3. йңҖиҰҒжүӢе·Ҙз®ЎзҗҶ vendor зӣ®еҪ•дёӢзҡ„ Go дҫқиө–еҢ…

еӣ жӯӨпјҢGo ж ёеҝғеӣўйҳҹе°Ҷ Go жһ„е»әзҡ„йҮҚзӮ№иҪ¬з§»еҲ°еҰӮдҪ•и§ЈеҶіеҢ…дҫқиө–з®ЎзҗҶдёҠ

Go Module

  1. д»Һ Go 1.11 ејҖе§ӢпјҢеј•е…ҘдәҶ Go Module жһ„е»әжЁЎејҸпјҢе№¶еңЁ Go 1.16 жҲҗдёәй»ҳи®Өзҡ„жһ„е»әжЁЎејҸ
  2. дёҖдёӘ Go Module жҳҜдёҖдёӘ Go еҢ…зҡ„йӣҶеҗҲ
    • Module жҳҜжңүзүҲжң¬зҡ„пјҢеӣ жӯӨ Module дёӢзҡ„еҢ…д№ҹжҳҜжңүзүҲжң¬зҡ„
    • Module е’ҢиҝҷдәӣеҢ…з»„жҲҗдёҖдёӘзӢ¬з«Ӣзҡ„зүҲжң¬еҚ•е…ғпјҢдёҖиө·жү“еҢ…гҖҒеҸ‘еёғгҖҒеҲҶеҸ‘
  3. еңЁ Go Module жЁЎејҸдёӢпјҢйҖҡеёёдёҖдёӘд»Јз Ғд»“еә“еҜ№еә”дёҖдёӘ Go Module
    • дёҖдёӘ Go Module зҡ„йЎ¶зә§зӣ®еҪ•дёӢдјҡж”ҫзҪ®дёҖдёӘ go.mod ж–Үд»¶пјҢдёҖдёҖеҜ№еә”
  4. go.mod ж–Үд»¶жүҖеңЁзҡ„йЎ¶еұӮзӣ®еҪ•д№ҹиў«з§°дёә Module зҡ„ж №зӣ®еҪ•
    • Module ж №зӣ®еҪ•д»ҘеҸҠе®ғзҡ„еӯҗзӣ®еҪ•зҡ„жүҖжңү Go еҢ…йғҪеҪ’еұһдәҺиҝҷдёӘ Go Module
    • иҝҷдёӘ Module д№ҹиў«з§°дёә Main Module

101

йҖҡиҝҮ go mod init еҲӣе»ә go.mod ж–Үд»¶пјҢе°ҶеҪ“еүҚйЎ№зӣ®еҸҳжҲҗдёҖдёӘ Go Module
Go йЎ№зӣ®еҸҜд»ҘеңЁд»»ж„Ҹи·Ҝеҫ„пјҢдёҚдёҖе®ҡиҰҒеңЁ GOPATH/src

go.mod ж–Үд»¶зҡ„з¬¬дёҖиЎҢеЈ°жҳҺдәҶ module pathпјҢжңҖеҗҺдёҖиЎҢдёә Go зүҲжң¬жҢҮзӨәз¬Ұ

  1. йҖҡиҝҮ go mod tidy иҮӘеҠЁжӣҙж–°еҪ“еүҚ Go Module зҡ„дҫқиө–е’Ңж ЎйӘҢе’Ң
  2. жү«жҸҸ Go жәҗз ҒпјҢиҮӘеҠЁжүҫеҮәйЎ№зӣ®дҫқиө–зҡ„еӨ–йғЁ Go ModuleпјҢдёӢиҪҪзӣҙжҺҘдҫқиө–е’Ңй—ҙжҺҘдҫқиө–е№¶жӣҙж–°жң¬ең°зҡ„ go.mod
  3. еҸҜд»ҘйҖҡиҝҮ $GOPROXY жқҘеҠ йҖҹз¬¬дёүж–№дҫқиө–пјҲGo Moduleпјүзҡ„дёӢиҪҪ
  4. дёӢиҪҪзҡ„дҫқиө–дјҡиў«ж”ҫзҪ®еңЁжң¬ең° Module зј“еӯҳи·Ҝеҫ„пјҢй»ҳи®Өдёә $GOPATH[0]/pkg/mod
    • д»Һ Go 1.15 ејҖе§ӢпјҢеҸҜд»ҘйҖҡиҝҮ GOMODCACHE жқҘиҮӘе®ҡд№ү Module зҡ„зј“еӯҳи·Ҝеҫ„
  5. go.sum еӯҳж”ҫзү№е®ҡзүҲжң¬зҡ„ Module еҶ…е®№зҡ„е“ҲеёҢеҖј - е®үе…ЁжҺӘж–Ҫ
    • еҰӮжһңжҹҗдёӘ Module зҡ„зү№е®ҡзүҲжң¬йңҖиҰҒиў«еҶҚж¬ЎдёӢиҪҪпјҢйңҖиҰҒе…ҲйҖҡиҝҮж ЎйӘҢе’Ңж ЎйӘҢ
    • жҺЁиҚҗе°Ҷ go.mod е’Ң go.sum дёҖе№¶жҸҗдәӨеҲ°д»Јз Ғд»“еә“

жү§иЎҢ go build

  1. иҜ»еҸ– go.mod дёӯзҡ„дҫқиө–еҸҠзүҲжң¬дҝЎжҒҜ
  2. еңЁжң¬ең° Module зј“еӯҳи·Ҝеҫ„жүҫеҲ°еҜ№еә”зүҲжң¬зҡ„дҫқиө–пјҢжү§иЎҢзј–иҜ‘е’Ңй“ҫжҺҘ

е·ҘдҪңеҺҹзҗҶ

иҜӯд№үеҜје…ҘзүҲжң¬

Semantic Import Versioning

  1. еңЁ Go Module жһ„е»әжЁЎејҸдёӢпјҢзүҲжң¬еҸ·пјҡvX.Y.Z
  2. Go е‘Ҫд»Өе’Ң go.mod ж–Үд»¶йғҪдҪҝз”ЁдёҠиҝ°з¬ҰеҗҲиҜӯд№үзүҲжң¬и§„иҢғзҡ„зүҲжң¬еҸ·жқҘжҸҸиҝ° Go Module зүҲжң¬
  3. еҖҹеҠ©иҜӯд№үзүҲжң¬и§„иҢғ
    • Go е‘Ҫд»ӨеҸҜд»ҘзЎ®е®ҡеҗҢдёҖдёӘ Module зҡ„дёӨдёӘзүҲжң¬еҸ‘еёғзҡ„е…ҲеҗҺж¬ЎеәҸпјҢд»ҘеҸҠжҳҜеҗҰе…је®№
  4. е…је®№жҖ§
    • дё»зүҲжң¬еҸ·зҡ„дёҚеҗҢзүҲжң¬жҳҜзӣёдә’дёҚе…је®№зҡ„
    • еңЁдё»зүҲжң¬еҸ·зӣёеҗҢзҡ„жғ…еҶөдёӢпјҢж¬ЎзүҲжң¬еҸ·йғҪжҳҜеҗ‘еҗҺе…је®№ж¬ЎзүҲжң¬еҸ·е°Ҹзҡ„зүҲжң¬
    • иЎҘдёҒзүҲжң¬еҸ·дёҚеҪұе“Қе…је®№жҖ§
  5. еҰӮжһңеҗҢдёҖдёӘеҢ…зҡ„ж–°ж—§зүҲжң¬жҳҜе…је®№зҡ„пјҢйӮЈд№Ҳе®ғд»¬зҡ„ import path жҳҜзӣёеҗҢзҡ„
    • logrus v1.8.1 е’Ң logrus v1.7.0 жҳҜе…је®№зҡ„пјҢйғҪжҳҜ v1 зүҲжң¬
    • import path йғҪдёә import "github.com/sirupsen/logrus"
  6. е°Ҷдё»зүҲжң¬еҸ·еј•е…ҘеҲ° import path дёӯ - иҜӯд№үеҜје…ҘзүҲжң¬пјҲGo еҲӣж–°пјү
    • еҸ‘еёғ logrus v2.0.0
    • import path дёә import "github.com/sirupsen/logrus/v2"
  7. v0.y.z
    • v0.y.z з”ЁдәҺйЎ№зӣ®еҲқе§ӢејҖеҸ‘йҳ¶ж®өзҡ„зүҲжң¬еҸ·пјҢAPI жҳҜдёҚзЁіе®ҡзҡ„
    • Go Module е°Ҷ v0 е’Ң v1 еҒҡеҗҢзӯүеҜ№еҫ…пјҢйҷҚдҪҺејҖеҸ‘дәәе‘ҳзҡ„еҝғжҷәиҙҹжӢ…

йҖҡиҝҮеңЁ import path дёӯеј•е…Ҙдё»зүҲжң¬еҸ·зҡ„ж–№ејҸпјҢжқҘеҢәеҲ«еҗҢдёҖдёӘеҢ…зҡ„дёҚе…је®№зүҲжң¬

жңҖе°ҸзүҲжң¬йҖүжӢ©

Minimal Version Selection

Go йҖүеҮәз¬ҰеҗҲж•ҙдҪ“иҰҒжұӮзҡ„жңҖе°ҸзүҲжң¬пјҢеҚі v1.3.0пјҢжңүеҠ©дәҺе®һзҺ° Reproducible Build

еҲҮжҚў

  1. Go 1.11 еј•е…Ҙ Go Module жһ„е»әжЁЎејҸ
    • жӯӨж—¶пјҢGOPATH жһ„е»әжЁЎејҸе’Ң Go Module жһ„е»әжЁЎејҸеҗ„иҮӘзӢ¬з«Ӣе·ҘдҪңпјҢеҸҜд»ҘйҖҡиҝҮзҺҜеўғеҸҳйҮҸ GO111MODULE жқҘеҲҮжҚў
  2. Go 1.16пјҢGo Module жһ„е»әжЁЎејҸжҲҗдёәдәҶй»ҳи®Өжһ„е»әжЁЎејҸ
    • еҗҺз»ӯзүҲжң¬еҸҜиғҪдјҡеҪ»еә•з§»йҷӨ GOPATH жһ„е»әжЁЎејҸпјҢGo Module жһ„е»әжЁЎејҸе°ҶжҲҗдёә Go е”ҜдёҖзҡ„ж ҮеҮҶжһ„е»әжЁЎејҸ
