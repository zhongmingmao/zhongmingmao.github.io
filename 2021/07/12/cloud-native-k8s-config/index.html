жіЁж„ҸдәӢйЎ№

  1. ConfigMap ж–Үд»¶еӨ§е°Ҹзҡ„йҷҗеҲ¶пјҡ1MBпјҲEtcdпјү
  2. Pod еҸӘиғҪеј•з”ЁзӣёеҗҢ Namespce дёӯзҡ„ ConfigMap
  3. Pod еј•з”Ёзҡ„ ConfigMap дёҚеӯҳеңЁж—¶пјҢPod ж— жі•еҲӣе»әжҲҗеҠҹ
  4. дҪҝз”Ё valueFrom д»Һ ConfigMap жқҘй…ҚзҪ®зҺҜеўғеҸҳйҮҸж—¶
    • ConfigMap дёӯжҹҗдәӣ Key иў«и®Өдёәж— ж•ҲпјҢиҜҘзҺҜеўғеҸҳйҮҸдёҚдјҡиў«жіЁе…Ҙе®№еҷЁпјҢдҪҶ Pod еҸҜд»ҘжӯЈеёёеҲӣе»ә
  5. еҸӘжңүйҖҡиҝҮ Kubernetes API еҲӣе»әзҡ„ Pod жүҚиғҪдҪҝз”Ё ConfigMap

Secret

з”ЁйҖ”

  1. еңЁйӣҶзҫӨдёӯпјҢз”ЁдәҺеӯҳеӮЁеҜҶз ҒгҖҒToken зӯүж•Ҹж„ҹдҝЎжҒҜ

  3. зұ»еһӢ

    • service-account-tokenпјҡз”ЁдәҺ Service Account иә«д»Ҫи®ӨиҜҒзҡ„ Secret
    • dockerconfigjsonпјҡз”ЁдәҺжӢүеҸ–з§Ғжңүд»“еә“й•ңеғҸзҡ„ Secret
    • bootstrap.tokenпјҡз”ЁдәҺиҠӮзӮ№жҺҘе…ҘйӣҶзҫӨж ЎйӘҢз”Ёзҡ„ Secret

Secret дё»иҰҒиў« Pod дҪҝз”ЁпјҢдёҖиҲ¬жҳҜйҖҡиҝҮжҢӮиҪҪ Volume зҡ„ж–№ејҸ

  2. Secret йҮҮз”Ё Base64 зј–з ҒпјҲзј–з Ғ != еҠ еҜҶпјҢеҸҜйҖүж–№жЎҲпјҡKubernetes + Vaultпјү
  3. з”ұдәҺ List/Watch дјҡиҺ·еҸ– Namespace дёӢжүҖжңүзҡ„ SecretпјҢе»әи®®дҪҝз”Ё Get

ServiceAccount

з”ЁйҖ”

  1. з”ЁдәҺи§ЈеҶі Pod еңЁйӣҶзҫӨдёӯзҡ„иә«д»Ҫи®ӨиҜҒй—®йўҳ
  2. и®ӨиҜҒдҪҝз”Ёзҡ„жҺҲжқғдҝЎжҒҜпјҢйҖҡиҝҮSecretпјҲkubernetes.io/service-account-tokenпјүиҝӣиЎҢз®ЎзҗҶ

е®һи·ө

  1. ServiceAccount.secretsпјҡжҢҮе®ҡ ServiceAccount дҪҝз”Ёе“ӘдёӘ SecretпјҢз”ұ Kubernetes иҮӘеҠЁж·»еҠ 
  2. ca.crtпјҡз”ЁдәҺж ЎйӘҢжңҚеҠЎз«Ҝзҡ„иҜҒд№ҰдҝЎжҒҜ
  3. tokenпјҡз”ЁдәҺ Pod зҡ„иә«д»Ҫи®ӨиҜҒ
  4. service-account.uidпјҡSecret е…іиҒ”зҡ„ ServiceAccount

Pod и®ҝй—® Kubernetes йӣҶзҫӨ

  1. Pod еҲӣе»әж—¶ Admission Controller дјҡж №жҚ®жҢҮе®ҡзҡ„ ServiceAccountпјҲй»ҳи®ӨжҳҜ defaultпјү жҠҠеҜ№еә”зҡ„ Secret жҢӮиҪҪеҲ°е®№еҷЁдёӯзҡ„еӣәе®ҡзӣ®еҪ•пјҲ**/var/run/secrets/kubernetes.io/serviceaccount**пјү
  2. еҪ“ Pod и®ҝй—®йӣҶзҫӨж—¶пјҢй»ҳи®ӨеҲ©з”Ё Secret дёӯзҡ„ token жқҘи®ӨиҜҒ Pod иә«д»ҪпјҲca.art з”ЁдәҺж ЎйӘҢжңҚеҠЎз«Ҝпјү
  3. Pod йҖҡиҝҮиә«д»Ҫи®ӨиҜҒеҗҺпјҢе…¶жқғйҷҗйңҖиҰҒйҖҡиҝҮ RBAC жқҘй…ҚзҪ®пјҢй»ҳи®ӨжҳҜ GET жқғйҷҗ

    • Guaranteed

    • BurstableпјҲеҸҜзҲҶпјү
      • Pod йҮҢиҮіе°‘жңүдёҖдёӘе®№еҷЁжңүеҶ…еӯҳжҲ– CPU зҡ„ requests

  2. еҪ“иҠӮзӮ№дёҠиө„жәҗдёҚи¶іж—¶пјҢй©ұйҖҗ Pod зҡ„дјҳе…Ҳзә§йЎәеәҸпјҡBestEffortгҖҒBurstableгҖҒGuaranteed

SecurityContext

  1. дё»иҰҒз”ЁдәҺйҷҗеҲ¶е®№еҷЁиЎҢдёәпјҢд»ҺиҖҢдҝқйҡңзі»з»ҹе’Ңе…¶д»–е®№еҷЁзҡ„е®үе…Ё
    • дёҚжҳҜ Kubernetes жҲ–иҖ… Container Runtime жң¬иә«зҡ„иғҪеҠӣпјҢиҖҢжҳҜдҫқиө–дәҺеҶ…ж ёжңәеҲ¶
  2. зә§еҲ«
    • е®№еҷЁзә§еҲ«зҡ„SecurityContextпјҡд»…еҜ№жҢҮе®ҡе®№еҷЁз”ҹж•Ҳ
    • Pod зә§еҲ«зҡ„SecurityContextпјҡеҜ№жҢҮе®ҡ Pod дёӯзҡ„жүҖжңүе®№еҷЁз”ҹж•Ҳ
    • Pod Security PoliciesпјҡеҜ№йӣҶзҫӨеҶ…жүҖжңү Pod з”ҹж•Ҳ

InitContainer

  1. InitContainer vs жҷ®йҖҡ Container
    • InitContainer дјҡе…ҲдәҺжҷ®йҖҡ Container еҗҜеҠЁжү§иЎҢпјҢзӣҙеҲ°жүҖжңү InitContainer жү§иЎҢжҲҗеҠҹеҗҺпјҢжҷ®йҖҡ Container жүҚдјҡиў«еҗҜеҠЁ
    • Pod дёӯеӨҡдёӘ InitContainer д№Ӣй—ҙжҢүйЎәеәҸеҗҜеҠЁпјҢиҖҢ Pod дёӯеӨҡдёӘжҷ®йҖҡ Container жҳҜе№¶иЎҢеҗҜеҠЁ
    • InitContainer жү§иЎҢжҲҗеҠҹеҗҺе°ұз»“жқҹйҖҖеҮәпјҢиҖҢжҷ®йҖҡ Container дјҡдёҖзӣҙжү§иЎҢжҲ–иҖ…йҮҚеҗҜпјҲrestartPolicy != Neverпјү
  2. з”ЁйҖ”
    • дёҖиҲ¬з”ЁдәҺжҷ®йҖҡ ContainerеҗҜеҠЁеүҚзҡ„еҲқе§ӢеҢ–жҲ–иҖ…еүҚзҪ®жқЎд»¶жЈҖйӘҢ
