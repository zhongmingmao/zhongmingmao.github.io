Java Feature - JShell

Created2024-10-08|JavaFeature
|Word Count:632|Reading Time:2mins

概述

  1. JShell 在 JDK 9 中正式发布
  2. JShell API 和工具提供了一种在 JShell 状态下交互式评估 Java 编程语言的声明、语句和表达式的方法
  3. JShell 的状态包括不断发展的代码和执行状态
  4. 为了快速调查和编码，语句和表达式不需要出现在方法中，变量和方法也不需要出现在类中
  5. JShell 在验证简单问题时，比 IDE 更高效

可覆盖

  1. JShell 支持变量的重复声明

表达式

  1. 在 Java 程序中，语句是最小的可执行单位，而表达式并不能单独存在
