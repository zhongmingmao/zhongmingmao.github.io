|Word Count:1k|Reading Time:5mins

函数

在 Python 中，函数是一等公民，函数是对象，可以将函数赋予变量

将函数赋值给变量

将函数当成函数参数传递给另一个函数

嵌套函数

函数可以返回函数对象 - 闭包

函数装饰器

朴素版本

优雅版本 - @ 是语法糖，相当于 greet=my_decorator(greet)，提高代码可读性

带有参数的装饰器

  1. 将 *args 和 **kwargs 作为装饰器内部函数 wrapper() 的函数参数
  2. *args 和 **kwargs - 表示接受任意数量和类型的参数

带有自定义参数的装饰器

greet() 函数被装饰后，函数的元信息变了

使用内置的装饰器 @functools.wraps - 保留原函数的元信息 - 将原函数的元信息拷贝到对应的装饰器函数里面

类装饰器

类装饰器主要依赖于函数 __call__()，每调用类的实例时，函数 __call__() 都会被执行一次

嵌套装饰器

执行顺序为从里到外

缓存

@lru_cache 会缓存进程中的函数参数和结果，当缓存满了以后，会删除 least recenly used 数据

小结

  1. 类似于 Spring AOP
  2. 横切面逻辑可以基于函数或者类（本质也是基于 __call__() 函数，但可以通过类属性记录更多信息）
  3. 装饰器 - 通过装饰器函数来修改原函数的一些功能，而不需要修改原函数代码，相当于一种增强

Decorators is to modify the behavior of the function through a wrapper so we don’t have to actually modify the function.
